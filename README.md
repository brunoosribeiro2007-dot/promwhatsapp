# Promoções de Casa - Link na Bio 🏠

Landing page elegante e minimalista estilo "link na bio", inspirada no design do Mercado Livre, focada em divulgar grupos de promoções (WhatsApp/Telegram).

## ✨ Funcionalidades
- 🎨 Design premium com cores do Mercado Livre (Amarelo & Branco).
- 🌊 Fundo animado com gradiente suave.
- 📱 Totalmente responsivo para dispositivos móveis.
- 🔗 Botões com efeitos de hover e ícones modernos.
- ⚡ Carregamento ultra-rápido (Vanilla HTML/CSS).

## 🚀 Como executar localmente
1. Clone este repositório.
2. Abra o arquivo `index.html` em qualquer navegador.

## 🛠️ Personalização
Para alterar os links dos grupos, edite as tags `<a>` no arquivo `index.html`.
